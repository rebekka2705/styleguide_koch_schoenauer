Links

Für unser Projekt ist ein Standardlink und ein Disabledlink ausreichend.
Standardlink
Disabledlink

HTML

         
            <a class="link" href="#">Standardlink</a>
            <a class="link link--disabled" href="#">Disabledlink</a>

SCSS

         
            .link {
               font-family: $font-mmp;
               font-size: 1.2rem;
               font-weight: bold;
               text-decoration: none;
               color: black;
               padding: 1rem;
               margin:1rem;
               
               &:hover {
                  background: $lightgreen;
                  color: white;
                  border-radius: 6px;
               }
               
               &--disabled {
                  color: grey;
               
                  &:hover{
                     background: none;
                     color: grey;
                     cursor: default;
                  }
               }
            }

Buttons

Unser Primary-Button verwendet unsere Secondary Textcolor white

HTML

       
         <button class="button">Standardbutton</button>
         <button class="button button--small">Kleiner Button</button>

SCSS

      
         .button {
            background: $darkgreen;
            border: none;
            border-radius: 10px;
            font-family: $font-mmp;
            font-weight: bold;
            font-size: $button;
            padding-top: 1rem;
            padding-bottom: 1rem;
            padding-left: 1rem;
            padding-right: 1rem;
            margin: auto;
            text-decoration: none;
            color: white;
          
            &:hover {
              transform: scale(1.1);
              transition-duration: 0.2s;
            }
          
            &--small {
              font-size: $button-small;
            }
          }
