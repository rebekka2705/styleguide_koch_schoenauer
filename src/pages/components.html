Form Elemente

Es gibt 2 verschiedene Arten von Form Elementen. Input Text Felder werden zum Beispiel für die Eingabe von Start und Ziel des Weges eingesetzt. Mithilfe von Radio Buttons, welche durch Bilder ersetzt werden, kann das gewünschte Verkehrsmittel ausgewählt werden. Das Auto wird standardgemäß ausgewählt.



HTML

         
            <form class="chapter__form">
               <input class="chapter__form__text" type="text"
                name="start" placeholder="Start"><br>
               <input class="chapter__form__text" type="text"
                name="end" placeholder="Ziel"><br>

            <label >
               <input class="chapter__form__radio" type="radio"
                name="transport-icon" value="small" checked>
               <img class="chapter__form__radio__image" 
               src=<%= require('./../images/car.svg') %>
            </label>
    
            <label>
               <input class="chapter__form__radio" type="radio" 
               name="transport-icon" value="big">
               <img class="chapter__form__radio__image" 
               src=<%= require('./../images/train.svg') %>
            </label>
            </form>

CSS

         
            .chapter__form {

               width: 80%;
               margin-left: 10%;
           
               .chapter__form__text {
                   font-family: $font-mmp;
                   font-size: 1.44rem;
                   color: black;
                   font-weight: bold;
                   border: 2px solid $lightgreen;
                   width: 100%;
                   height: 40px;
                   padding-left: 2%;
                   margin-bottom: 10px;
                   box-shadow: 2px 1px 1px $darkgreen;
                   border-radius: 10px;
               }
           
               .chapter__form__text:focus {
                   color: white;
                   font-weight: bold;
                   background-color: $lightgreen;
               }
           
               .chapter__form__radio { 
                   position: absolute;
                   opacity: 0;
                   width: 0;

                   &__image {
                       width: 50px;

                       border-width: 2.5px;
                       border-style: solid;
                       margin-right: 5px;
                   }
           
                   &__image:hover {
                       background-color: $lightgreen;
                   }
               }
                 
               .chapter__form__radio + .chapter__form__radio__image {
                   cursor: pointer;
               }
           
               .chapter__form__radio:checked + .chapter__form__radio__image {
                   background-color: $lightgreen;

Buttons

Unser Buttons sind durch dunklen Hintergrund gekennzeichnet sowie weißer Textfarbe. Neben dem Standardbutton verwenden wir in unserem Projekt den Button in small.

HTML

      
         <button class="button">Standardbutton</button>
         <button class="button button--small">Kleiner Button</button>

SCSS

      
         .button {
            width: 80%;
            margin-left: 10%;
            background: $darkgreen;
            border: none;
            border-radius: 10px;
            font-family: $font-mmp;
            font-weight: bold;
            font-size: $button;
            padding-top: 1rem;
            padding-bottom: 1rem;
            padding-left: 1rem;
            padding-right: 1rem;
            text-decoration: none;
            color: white;
            cursor: pointer;
          
            &:hover {
              transform: scale(1.1);
              transition-duration: 0.2s;
            }
          
            &--small {
              width: 40%;
              margin: 0 5%;
              font-size: $button-small;
            }
         }

Feedback

Feedback Elemente sind Pop-ups welche Hinweise, oder auch Entscheidungen beinhalten können. Sie enthalten ein Textelement, sowie einen Standardbutton oder zwei kleine Buttons.

Du hast dich erfolgreich ausgeloggt!

Willst du wirklich dein Profil löschen?

         <div class="mobile-size">
            <div class="feedback">
               <p class="feedback__text">
               Du hast dich erfolgreich ausgeloggt!</p>
               <button class="button">Schliessen</button>
            </div>
         </div>
